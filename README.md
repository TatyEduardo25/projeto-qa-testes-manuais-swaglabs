## 📁 Documentação e Entregas

Toda a documentação gerada ao longo das etapas do projeto está disponível diretamente na raiz deste repositório em formato PDF:

   * Definição das colunas do board ágil para as histórias da Sprint.

3. **[Mapa Mental de Cenários de Teste](./03-mapa-mental.pdf)**
   * Mapeamento visual e estruturado de cenários de teste positivos, negativos e de interface.

4. **[Plano e Casos de Teste (Zephyr Scale)](./04-casos-de-teste.pdf)**
   * 2 Casos de Teste detalhados utilizando a técnica tradicional Step-by-Step.

   * Criação e consolidação dos testes dentro de um Ciclo de Execução de Testes.
